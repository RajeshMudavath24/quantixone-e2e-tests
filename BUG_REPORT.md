# Bug Report - QuantixOne Testing

## BUG-RESP-001
**Title:** Horizontal scroll issue on mobile viewport  
**URL:** https://quantixone.com  
**Browser:** Chromium, WebKit  
**Viewport:** 375x667  

**Steps to Reproduce:**
1. Open the website in mobile view (375px width)
2. Observe the page layout

**Expected Result:**
Page should fit within the viewport without horizontal scrolling  

**Actual Result:**
Page width exceeds viewport causing horizontal scroll  

**Severity:** Medium  
**Status:** Open
